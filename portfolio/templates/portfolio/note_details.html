Project Title: Online Notes Sharing Platform

Platform Overview

This platform is designed for users to share notes online, making it easy for students and professionals to access, upload, and download notes on various subjects. Users can upload their notes, view others' uploads, and download files.

Features

  • User registration and secure login
  • Upload, edit, and delete notes
  • Download notes with permission control
  • Notification system for updates and interactions
  • Responsive design for cross-device accessibility

Technologies Used

  • Django (Backend Framework)
  • SQLite/MySQL (Database)
  • HTML/CSS and Tailwind CSS (Frontend)
  • JavaScript for interactive elements

Developing this online notes-sharing platform involved addressing challenges related to file upload management, permission control, and secure access.
