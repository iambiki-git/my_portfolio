Project Title : FashionMart

Project Overview

This project is a modern e-commerce platform built with Django and custom CSS. It focuses on delivering a seamless shopping experience, Cash on delivery (COD) as a secure payment option and an attractive user interface.

Features

  • Interative UI
  • Similar Product Recommendation
  • Product Search and Filtering
  • Admin Dashboard for Product Management

Technologies Used

  • Django (Backend)
  • Custom CSS (Styling)
  • JavaScript (Frontend)
  • SQLite (Database)
