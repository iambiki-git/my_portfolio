Project Title: My Django Blog

Blog Overview

This blog platform allows users to share their blog posts on various topics or categories. Users can upload, edit, and delete their posts, while also having the ability to like posts. If a user is not logged in, they can view the post title, short-description, author name of the blog post but cannot view details of that post. Additionally, there is a notification system where users are notified when they like a post, have a post deleted, or when new content is added.

Features

  • User registration and authentication
  • Upload, edit, and delete blog posts
  • Like posts and view notifications
  • View blog details without login (limited interaction)
  • Responsive user profile

Technologies Used

  • Django (Backend Framework)
  • SQLite (Database)
  • HTML/Tailwind CSS (Frontend)
  • JavaScript (Frontend Interactivity)
